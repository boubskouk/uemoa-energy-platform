# Images Assets

Ce dossier contient les images et logos de l'application.

## Logo UEMOA

Pour ajouter le logo de l'UEMOA :

1. Placez votre fichier logo ici avec le nom `uemoa-logo.png` ou `uemoa-logo.svg`
2. Le logo sera automatiquement utilisé dans l'application

## Formats acceptés

- PNG (recommandé pour les logos avec transparence)
- SVG (recommandé pour la qualité à toutes les tailles)
- JPG

## Taille recommandée

- Logo : 200x200px minimum
- Format : PNG ou SVG avec fond transparent
